# 📊 Student Grade Tracker

A simple Java app I built to track student grades, calculate scores, and generate reports — all from the console.

---

## 👋 What is this?

Ever wanted a quick way to manage student grades without opening Excel?
That's exactly why I built this. You just run the program, add students,
enter their scores, and it does all the math for you — averages, highest,
lowest, letter grades, everything.

---

## ✨ What can it do?

- Add as many students as you want
- Enter multiple grades per student
- Auto-calculates average, highest & lowest score
- Gives letter grades (A / B / C / D / F)
- Shows a full class summary report
- Visual progress bar per student
- Remove students anytime
- Won't let you enter invalid scores (only 0–100 allowed)

---

## ▶ How to run it?

Make sure you have **Java JDK 17+** installed, then:

```bash
javac StudentGradeTracker.java
java StudentGradeTracker
```

Or just open it in **IntelliJ IDEA** and hit the ▶ play button.

---

## 📂 What's inside?

```
src/
├── Student.java               → holds student name & grades
└── StudentGradeTracker.java   → runs the whole app
```

Two files. That's it.
